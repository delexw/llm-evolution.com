# LLM Model Evolution 2024–2026

An interactive reference tracking 244 large language model releases across 25 families and 13 providers — from April 2024 to March 2026.

**Live site:** [llm-evolution.com](https://www.llm-evolution.com/)

## Why

One day I realized the AI emails I'd been subscribing to since 2024 had piled up and I basically lost track of what was in them.
AI models are developing so fast and I wanted to rewind all that information so I could stay on track and pick the right model for the right job.
The emails were really useful — they covered each major model in release order.
So I used an agent with the GWS CLI to go through hundreds of them and extract the model updates.
The data goes up to March 2026 — that's where my email archive ends for now.
My emails keep coming in daily, so the next update will happen someday when I get around to it.

## What's inside

- Complete catalog of 244 models with parameters, context windows, release dates, and tier classifications
- Interactive timeline filterable by provider
- Context window growth chart (8K → 10M tokens)
- Provider activity and release velocity charts
- Model size distribution breakdown
- Five defining trends analysis
